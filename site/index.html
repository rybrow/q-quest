About Q-Quest!

Q-Quest! is an immersive turn-based RPG that combines classic gameplay with modern AI-generated visuals. Navigate through various levels, engage in strategic combat, collect loot, and defeat powerful enemies.

Built with the Amazon Q CLI and powered by Amazon Bedrock and Amazon Nova, this game showcases the potential of AI in game development. Each character portrait and map icon is uniquely generated using AI models, creating a rich and diverse game world.

Game Features

Multiple Levels

Explore the Wizard's Tower, Ancient Dungeon, and City Sewers, each with unique enemies and challenges.

Turn-Based Combat

Engage in strategic combat with various enemies, using attacks, defense, and healing abilities.

Character Progression

Collect loot from defeated enemies to improve your character's abilities and equipment.

AI-Generated Art

Enjoy unique character portraits and map icons generated by Amazon Bedrock AI models.

Intuitive Controls

Easy to learn controls make the game accessible for players of all experience levels.
